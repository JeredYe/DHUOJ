问题描述

输入两个实数a和b，计算a/b的值，并输出。
如果b为0，则输出error，否则输出结果，保留两位小数。

 

C语言代码如下：
#include <stdio.h>
int main()
{
    double a,b,c;
    while(scanf("%lf%lf", &a, &b) != EOF)
    {
    if (b!=0)
    {
        c=a/b;
        printf("%.2f\n", c);
    }
    else
        printf("error\n");
    }

    return 0;
}

将以上代码复制到提交框中，运行结果将为AC（正确），可看到所有测试用例运行都正确。
